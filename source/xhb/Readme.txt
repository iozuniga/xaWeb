xaWeb xHarbour:
==============

Un módulo de la librería [xharbour] de contribuciones de Harbour es necesario para que funcione correctamente la clase ZWebDataSource. Dicho módulo es el de soporte de XML. 

xaWeb sólo incluye estos módulos en su librería principal en su versión para Linux, ya que para Windows, se incluyen directamente al incorporar Xailer.lib al proyecto.

-------------------------------------------------------------------------------

A module from Harbour's [xharbour] library of contributions is required for the ZWebDataSource class to work properly. This module is the XML support module.

xaWeb only includes these modules in its main library for Linux; for Windows, they are included directly when you add Xailer.lib to your project.
